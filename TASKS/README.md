# Задачи проекта ОКАК

Этот каталог — рабочий бэклог команды на MVP. Все задачи описаны так, чтобы сразу было понятно, что делать, где читать документацию, где писать код и какой результат считать завершением.

## Формат задачи

- Название задачи
- Зачем нужна
- Что обязательно прочитать
- Где в коде работать
- Что должно получиться на выходе
- Зависит от
- Можно делать параллельно
- Крайний срок

## Как работаем параллельно

- Карта зависимостей: `TASKS/Зависимости_и_параллельность.md`
- Все, что не связано прямой зависимостью, делаем параллельно.
- Если задача заблокирована, фиксируем блокер и переключаемся на параллельную задачу.

## Команда

- Костя: backend-основа, авторизация, проекты, заметки, контроль доступа.
- Андрей: задачи, файлы, теги, поиск, технические проверки API.
- Миша: AI/LLM-функции, интеграция провайдеров, fallback-режим.
- Маша: подключение frontend к backend API, основные пользовательские экраны.
- Даша: UI-состояния, формы, ручная регрессия, вспомогательная frontend-разработка.
- Александр: Electron-версия приложения, desktop UX, сборка и запуск.
